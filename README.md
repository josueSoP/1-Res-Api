### Pasos Para Correr el Proyecto
1. Clonar Repositorio

2. Instalar Dependencias
    - `npm install`

3. Crear un archivo .ENV para las variables de entorno
   
3. Iniciar Servidor Local
    - `npm run dev` 

4. Ruta de servidor del Usuario
    - http://localhost:3000/
